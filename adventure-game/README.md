# poem-finder
bash adventure game

`sh adventure-game.sh` to play

### walkthrough
here's a list of commands that will get you through the game:
```
<your name>
yes
goto 4
open envelope
take poem piece
goto 0
goto 5
open book
take poem piece
goto 0
goto 7
use piano
take poem piece
<poem title>
```

poem by e.e. cummings
